# 快速解决方案 - 只需 3 步

## 🚨 您的问题：WSL 未安装 Ubuntu

您的 WSL 已经安装，但缺少 Linux 发行版（Ubuntu）。

---

## ✅ 解决方法（选择一个）

### 方法一：使用自动脚本（最简单）⭐

1. **双击运行** → `安装Ubuntu.bat`
2. 等待安装完成（约 2-5 分钟）
3. 首次启动会要求设置**用户名**和**密码**（记住密码！）
4. 完成后，双击运行 → `一键编译-v2.bat`

---

### 方法二：手动安装

**在 PowerShell（管理员）中运行：**

```powershell
wsl --install -d Ubuntu-24.04
```

等待安装完成后，运行：
```powershell
wsl
```

首次启动设置用户名和密码，然后关闭窗口，运行 `一键编译-v2.bat`

---

## ❓ 常见问题

### Q: 设置什么用户名和密码？
随便设置，例如：
- 用户名: `user` 或 `ubuntu`
- 密码: 任意（输入时不显示，是正常的）

### Q: 忘记密码怎么办？
不影响使用，编译时会自动跳过需要密码的步骤。

### Q: 安装失败？
1. 确保网络连接正常
2. 以**管理员身份**运行 PowerShell
3. 重试命令: `wsl --install -d Ubuntu-24.04`
